# Microcomputer Principles Lab

微机原理与接口技术学习仓库，内容包括处理器结构、汇编语言、存储器与 I/O 地址译码、中断系统以及 8255、8253/8254、8259 等接口芯片。

## 目录

```text
notes/
assembly/
interface-chips/
├── 8255/
├── 8253-8254/
└── 8259/
memory-and-io/
simulations/
reports/
learning-log/
```

## 学习目标

- 理解 CPU、存储器、I/O 和总线之间的关系
- 掌握基础汇编程序结构和寻址方式
- 理解中断响应和中断控制器
- 能够完成接口芯片的地址分配与程序控制
