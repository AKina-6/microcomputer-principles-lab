# Contributing

本仓库主要用于个人学习记录和嵌入式作品集建设。

## 代码要求

1. 硬件相关代码与业务逻辑尽量分离。
2. 每个实验提供独立 README。
3. 提交前完成编译和基础运行验证。
4. 注释说明设计目的，避免逐行翻译代码。
5. 不提交编译产物、密钥和第三方受限文件。

## Commit Message 示例

```text
feat: add EXTI button interrupt demo
docs: explain I2C address and ACK sequence
fix: correct Modbus CRC byte order
test: add alarm threshold cases
```
