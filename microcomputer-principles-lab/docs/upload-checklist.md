# 每日上传检查表

提交前检查：

- [ ] 当日代码可以编译
- [ ] README 或学习日志已更新
- [ ] 已记录硬件连接
- [ ] 已记录实验结果
- [ ] 未提交编译产物
- [ ] 未提交密钥、账号或服务器地址
- [ ] 图片经过压缩且命名清晰
- [ ] Commit Message 能说明本次学习成果

推荐提交格式：

```bash
git add .
git commit -m "feat: add GPIO output experiment"
git push
```

常用类型：

- `feat`：增加功能或实验
- `fix`：修复错误
- `docs`：更新文档
- `test`：增加测试
- `refactor`：重构代码
- `chore`：仓库维护
